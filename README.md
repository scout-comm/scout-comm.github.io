# SCOUT project website (double-blind)

This folder is the source for **https://scout-comm.github.io/** (RLC submission).

- **Code repo:** https://github.com/scout-comm/scout  
- No author names or identifying links on the site (double-blind).

## Deploy

1. Create a **new repo** on the **scout-comm** GitHub account named exactly: **scout-comm.github.io**
2. Push this folder to that repo (e.g. clone the empty repo, copy these files in, push to `main`).
3. GitHub Pages will serve the site at **https://scout-comm.github.io/** (Settings → Pages → source: main branch).

## Edit before submission

- **index.html:** Replace placeholder title, abstract, and methodology with your content.
- **Paper link:** Update the `#` in the Paper button to your anonymous PDF or “Paper link will be added upon publication.”
- **BibTeX:** Add after acceptance.
- **Images:** Add any figures to `imgs/` and reference in the HTML.

## Template

Layout and styling follow the structure of `bcod-diffusion.github.io` (sidebar, sections, buttons). Add more sections or media as needed.
